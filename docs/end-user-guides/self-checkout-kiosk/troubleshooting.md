# Troubleshooting

Solutions to common issues you might encounter at the self-checkout kiosk.

---

## Quick Help

For immediate assistance at any time:
- **Press the "Help" button** on screen
- **Look for store associates** in the self-checkout area
- **Visit Customer Service** near the store entrance

---

## Scanning Issues

### Barcode Won't Scan

**Symptoms:** Item doesn't register when held up to scanner, no beep sound.

**Solutions:**

1. **Check scanner light**
   - The red scanning light should be visible
   - If not, the scanner may be disabled - tap the screen to activate

2. **Reposition the barcode**
   - Hold 4-8 inches from scanner
   - Ensure barcode is flat and facing the scanner
   - Try different angles

3. **Check barcode condition**
   - Smooth out wrinkled barcodes
   - Wipe off any moisture or debris
   - Remove protective plastic if covering barcode

4. **Use manual entry**
   - Tap "Enter SKU" on screen
   - Type the numbers below the barcode
   - See [Scanning Products](scanning-products.md) for details

5. **Request assistance**
   - Press "Help" if problems persist

### Item Not Found

**Symptoms:** "Item not found" or "SKU not recognized" message appears.

**Solutions:**

| Cause | Solution |
|-------|----------|
| Wrong SKU entered | Double-check the number and try again |
| New product | Item may not be in system - request help |
| Discontinued item | May need manual price entry by associate |
| Wrong barcode | Some items have multiple barcodes - try another |

**Steps:**
1. Verify you entered the correct SKU
2. Try scanning instead of manual entry (or vice versa)
3. Press "Help" for associate assistance

---

### Wrong Item Scanned

**Symptoms:** Different product appears than what you scanned.

**Solutions:**
1. **Remove the incorrect item** from your cart immediately
2. **Re-scan** the correct product
3. If it happens again, **request help** - the barcode may be mislabeled

## Cart Issues

### Can't Remove Item

**Symptoms:** Unable to delete an item from cart.

**Solutions:**

1. **Swipe method**
   - In cart panel, swipe left on the item
   - Tap "Remove" button that appears

2. **Full cart view**
   - Tap "Review Cart" to see full cart
   - Tap trash icon next to item
   - Confirm removal

3. **Quantity method**
   - Reduce quantity to 0 using minus button
   - Item should remove automatically

4. **Still stuck?**
   - Press "Help" - associate can remove items

---

### Price Doesn't Match Shelf

**Symptoms:** Scanned price differs from posted shelf price.

**What to do:**
1. **Note the discrepancy** - don't complete payment yet
2. **Press "Help"** to request a price check
3. **Show the associate** where you found the item
4. **They will adjust** if shelf price was correct

**Store policy:** Most stores honor the posted shelf price if there's a discrepancy.

---

### Missing Loyalty Discounts

**Symptoms:** Expected member discounts not applied.

**Checklist:**

| Check | Action |
|-------|--------|
| Account linked? | Verify your name appears on screen |
| Correct account? | Make sure it's your account, not someone else's |
| Item eligible? | Not all items qualify for loyalty discounts |
| Offer expired? | Some promotions have end dates |
| Minimum purchase? | Some discounts require minimum spend |

**Steps:**
1. Verify loyalty account is linked (your name should display)
2. Check if items are marked as "member price" on shelf
3. Press "Help" if discounts should apply but don't

## Payment Issues

### Card Declined

**Symptoms:** "Payment declined" or "Card not authorized" message.

**Common causes and solutions:**

| Cause | Solution |
|-------|----------|
| Insufficient funds | Use different card or reduce purchase |
| Daily limit reached | Contact your bank or use different card |
| Card expired | Check expiration date, use valid card |
| Security block | Bank may have flagged transaction - call them |
| Chip malfunction | Try swiping instead of chip insert |
| Wrong PIN | Re-enter PIN carefully |

**Steps:**
1. Try the same card again (sometimes works on retry)
2. Try a different payment method
3. Check your bank app for any alerts
4. Contact your bank if needed

---

### Card Reader Not Responding

**Symptoms:** Card reader doesn't react when card inserted/tapped.

**Solutions:**

1. **Check screen prompt**
   - Reader activates only when screen says "Insert/Tap Card"
   - Make sure you've selected a payment method first

2. **Try different method**
   - If tap doesn't work, try chip insert
   - If chip doesn't work, try magnetic swipe

3. **Clean card contacts**
   - Wipe chip with clean cloth
   - Wipe magnetic stripe

4. **Wait and retry**
   - Remove card completely
   - Wait 5 seconds
   - Try again

5. **Request help**
   - Reader may need reset by associate

### "Remove Card" Message Won't Clear

**Symptoms:** Screen keeps showing "Remove card" but card is already removed.

**Solutions:**
1. Check card slot - ensure card is fully removed
2. Look for a stuck card or foreign object
3. Tap screen to acknowledge
4. Press "Help" if message persists

## Screen Issues

### Screen Unresponsive

**Symptoms:** Tapping screen has no effect.

**Solutions:**

1. **Wait a moment**
   - System may be processing
   - Look for spinning/loading indicator

2. **Check for dialog boxes**
   - A popup may be waiting for input
   - Look for buttons to dismiss

3. **Try different areas**
   - Screen may have dead spots
   - Try tapping buttons in different locations

4. **Report to associate**
   - The kiosk may need to be reset
   - Move to another kiosk if available

---

### Session Timed Out

**Symptoms:** Transaction canceled, returned to welcome screen.

**What happened:** Kiosks automatically reset after 2 minutes of inactivity.

**What to do:**
1. Start a new transaction
2. Re-scan all items
3. Stay active by touching screen periodically during checkout

**Prevention:**
- The kiosk warns you at 30 seconds before timeout
- Tap "Continue" when you see the warning
- Don't step away mid-transaction

---

### Display Looks Wrong

**Symptoms:** Colors distorted, text cut off, or display garbled.

**Solutions:**
1. Try a different kiosk
2. Report the issue to store staff
3. The kiosk may need maintenance

## Loyalty Account Issues

### Can't Find My Account

**Symptoms:** Phone or email lookup returns "not found."

**Possible causes:**

| Issue | Solution |
|-------|----------|
| Typo in entry | Re-enter phone/email carefully |
| Different contact | Try alternate phone number or email |
| No account exists | Sign up at customer service |
| Account deactivated | Contact customer service |

**Steps:**
1. Try entering again (check for typos)
2. Try phone if email didn't work (or vice versa)
3. Skip for now, link later at customer service

---

### Wrong Account Linked

**Symptoms:** Someone else's name appears after lookup.

**Immediate action:**
1. **Tap "Not you?"** or **"Use Different Account"**
2. Enter your own information
3. Never use someone else's account

**Why this happens:**
- Previous customer may have used the same phone/email
- You may have entered number incorrectly

---

### Loyalty System Unavailable

**Symptoms:** "Service unavailable" when trying to link account.

**What to do:**
1. Try once more
2. Skip loyalty lookup - continue as guest
3. Save your receipt - customer service can add points later

---

## Receipt Issues

### Receipt Didn't Print

**Symptoms:** Printer didn't produce receipt.

**Solutions:**
1. Check if you selected "Email" or "No receipt"
2. Look in the receipt slot - it may be there
3. Request reprint from store associate (they can look up transaction)

**For returns/exchanges:** Customer service can look up your transaction by card used.

---

### Printer Out of Paper

**Symptoms:** "Printer error" or empty receipt slot.

**Solutions:**
1. Select "Email receipt" instead
2. Request printed receipt from customer service
3. Transaction is still complete - you can proceed

---

## Transaction Issues

### Transaction Stuck

**Symptoms:** Screen frozen, nothing happening for extended period.

**Wait times to expect:**
- Payment processing: up to 30 seconds
- Loyalty lookup: up to 10 seconds
- Item scanning: instant to 3 seconds

**If stuck longer:**
1. Wait 1 minute - system may recover
2. Press "Help" button
3. Do NOT leave the kiosk - associate will assist
4. Your items and payment status will be verified

---

### Accidentally Canceled

**Symptoms:** Transaction ended before completion.

**What to do:**
1. Start new transaction
2. Re-scan all items
3. Proceed through checkout again

**Preventing accidental cancel:**
- Read confirmation dialogs carefully
- "Cancel" always requires confirmation
- Keep transaction area clear of others

---

## Getting Help

### When to Press Help

Press the Help button when:
- You can't resolve an issue yourself
- An error message appears that you don't understand
- Payment fails repeatedly
- You need a price check
- The kiosk seems to malfunction

### What Happens When You Press Help

1. A **light may flash** above your kiosk
2. Your screen shows **"Help requested"**
3. An **associate will come** to assist
4. Continue your transaction when helped

### Help Response Time

- During normal hours: 1-3 minutes
- During busy periods: may take longer
- If urgent, approach the self-checkout attendant directly

## Error Codes

If you see an error code, note it for the associate:

| Code | Meaning |
|------|---------|
| E001 | Scanner communication error |
| E002 | Payment terminal error |
| E003 | Printer error |
| E004 | Network connection issue |
| E005 | Session expired |

Associates can resolve these issues quickly when they know the error code.
