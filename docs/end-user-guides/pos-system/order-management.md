## Best Practices

### Order Lookup

- Use order number when available (fastest)
- Verify customer identity before sharing details
- Check order status before promising changes

### Returns

- Inspect items before processing
- Document condition accurately
- Follow return policy timeframes
- Get manager approval when required

### Modifications

- Confirm changes with customer
- Re-send confirmation if contact info changes
- Note reason for modification
