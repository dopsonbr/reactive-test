# Contact Center Operations

Learn how to handle phone orders, remote payments, and contact center-specific workflows.

---

## Contact Center Overview

Contact center agents handle:
- Phone orders from customers
- Order inquiries and modifications
- Remote payment capture
- Customer service issues
- Delivery scheduling

---

## Starting a Phone Order

### New Phone Order Workflow

1. Answer call and greet customer
2. Look up or create customer profile
3. Enter items (manual entry common)
4. Configure delivery
5. Capture payment (card-not-present)
6. Confirm order details

## Contact Center Best Practices

### Communication

| Do | Don't |
|----|-------|
| Speak clearly | Rush through information |
| Confirm details | Assume understanding |
| Use customer's name | Be overly familiar |
| Offer alternatives | Just say "no" |
| Thank for patience | Blame other departments |

### Security

| Do | Don't |
|----|-------|
| Verify identity | Skip verification |
| Follow PCI rules | Write down card numbers |
| Note unusual requests | Process suspicious orders |
| Escalate when unsure | Guess at policies |

### Efficiency

| Do | Don't |
|----|-------|
| Use keyboard shortcuts | Over-rely on mouse |
| Pre-fill known data | Re-enter information |
| Use saved addresses | Type addresses repeatedly |
| Batch similar tasks | Context-switch constantly |
