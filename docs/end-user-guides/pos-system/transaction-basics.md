## Adding Items

### Barcode Scanning

The fastest way to add items:

1. Position the item's barcode toward the scanner
2. Scanner automatically reads and beeps
3. Item appears in cart with quantity 1
4. Scan same item again to increment quantity

![Scanning Items](images/scanning-item-wireframe.png)
*Scan barcodes for instant item entry*

**Scanning Tips:**
- Hold barcode 4-8 inches from scanner
- Ensure barcode is clean and undamaged
- Wait for beep before scanning next item
- Scanner works with UPC, EAN, and SKU barcodes

### Manual SKU Entry

When a barcode won't scan:

1. Click in the **SKU entry field** or press **F7**
2. Type the SKU number
3. Press **Enter** to add item
4. Add quantity prefix for multiple items (e.g., "3*SKU123")

### Adjusting Quantities

**Using +/- Buttons:**
1. Click **-** to decrease quantity
2. Click **+** to increase quantity
3. Minimum quantity is 1 (use remove to delete)

**Direct Entry:**
1. Click the quantity number
2. Type new quantity
3. Press Enter to confirm

**Keyboard:**
- Select item with arrow keys
- Press **+** to increase
- Press **-** to decrease
- Press **Delete** to remove

### Removing Items

To remove an item from the cart:

1. Click the **trash icon** (ğŸ—‘ï¸) on the item, or
2. Select item and press **Delete** key, or
3. Set quantity to 0

## Transaction Actions

### Linking a Customer

Add a customer to the transaction for:
- Loyalty points and discounts
- Purchase history
- Receipt email
- Returns without receipt

See [Customer Lookup](customer-lookup.md) for details.

### Changing Fulfillment

The default fulfillment is usually "Immediate" (customer takes items now). To change:

1. Click the **Fulfillment dropdown**
2. Select desired type (Pickup, Delivery, etc.)
3. Configure options if prompted

See [Fulfillment Options](fulfillment-options.md) for details.

### Applying Discounts

To apply markdowns or promotional discounts:

1. Select the item in cart
2. Click **"Apply Markdown"** or press **F6**
3. Select reason and enter amount
4. Manager approval may be required

See [Markdowns & Discounts](markdowns-discounts.md) for details.

---

## Suspending a Transaction

If you need to pause a transaction (customer forgot wallet, need manager, etc.):

1. Click **"Suspend"** or press **F5**
2. Enter a note (optional) to identify later
3. Transaction is saved and cart clears

## Proceeding to Payment

When ready to complete the transaction:

1. Review the cart and total
2. Ensure customer is linked (if applicable)
3. Verify fulfillment type is correct
4. Click **"Pay"** or press **F8**

See [Payment Processing](payment-processing.md) for completing the sale.

## Transaction Tips

### Speed Tips

| Tip | Benefit |
|-----|---------|
| Scan continuously | Scanner handles rapid input |
| Use quantity prefix | "5*SKU" faster than 5 scans |
| Learn shortcuts | F2 for new, F8 for pay |
| Use product search | Faster than finding barcode |

### Accuracy Tips

| Tip | Benefit |
|-----|---------|
| Verify quantities | Check cart before payment |
| Link customer first | Ensures discounts apply |
| Check fulfillment | Avoid wrong delivery type |
| Review before pay | Catches mistakes early |
