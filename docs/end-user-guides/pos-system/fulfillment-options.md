# Fulfillment Options

Learn about the different ways customers can receive their purchases.

---

## Fulfillment Types Overview

| Type | Description | Customer Gets Items |
|------|-------------|---------------------|
| **Immediate** | Customer takes items now | At checkout |
| **Pickup** | Customer picks up later | At scheduled time |
| **Will-Call** | Items held for pickup | During specified window |
| **Delivery** | Shipped to address | Delivered to door |
| **Installation** | Professional setup | By appointment |

---

## Immediate (Carryout)

The default option for in-store purchases.

### When to Use
- Standard walk-in purchases
- Customer is taking items with them
- No scheduling needed

### How to Use
1. Immediate is selected by default
2. Complete transaction normally
3. Customer takes items after payment

## Pickup (Buy Online, Pickup In-Store)

Customer purchases now, picks up later at this or another store.

### When to Use
- Customer can't carry items now
- Items need to be retrieved from back stock
- BOPIS (Buy Online Pickup In-Store) orders

### Configuring Pickup

1. Click **Fulfillment** dropdown
2. Select **"Pickup"**
3. Choose pickup location
4. Select pickup date/time

## Changing Fulfillment

### Before Payment

Fulfillment can be freely changed:
1. Click **"Change"** next to fulfillment
2. Select new type
3. Configure as needed

### After Payment

Fulfillment changes after payment require:
1. Manager authorization for some changes
2. Potential refund/charge for delivery fee differences
3. Order modification workflow

## Best Practices

### Ask the Right Questions

| Question | Determines |
|----------|------------|
| "Taking this with you today?" | Immediate vs other |
| "Delivery address same as billing?" | Address confirmation |
| "Preferred delivery date?" | Scheduling |
| "Need installation?" | Service add-ons |

### Common Scenarios

| Scenario | Recommended Fulfillment |
|----------|------------------------|
| Small items, customer present | Immediate |
| Large items, customer's vehicle | Pickup (load assist) |
| Furniture, appliances | Delivery |
| Electronics setup needed | Installation |
| B2B bulk order | Multi-fulfillment |
