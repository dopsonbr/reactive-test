# Offline POS Images

This directory contains wireframe images and screenshots for the Offline POS User Guide.

## Required Images

| Filename | Description | Used In |
|----------|-------------|---------|
| `offline-pos-overview-wireframe.png` | Main interface overview | index.md |
| `scan-screen-wireframe.png` | Scan screen layout | getting-started.md |
| `barcode-scan-wireframe.png` | Scanning demonstration | scanning-items.md |

## Image Guidelines

- **Format:** PNG preferred, JPEG acceptable
- **Size:** 800-1200px width recommended
- **Style:** Wireframe or screenshot with annotations
- **Naming:** `descriptive-name-wireframe.png` or `descriptive-name-screenshot.png`

## Creating Wireframes

Wireframes can be created using:
- Figma
- Excalidraw
- Balsamiq
- Draw.io

Keep wireframes simple and focused on the UI element being explained.

## Adding New Images

1. Create the image following guidelines above
2. Save to this directory with descriptive name
3. Reference in markdown: `![Alt Text](images/filename.png)`
4. Add to this README table
