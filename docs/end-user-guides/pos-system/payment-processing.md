# Payment Processing

Learn how to process payments across all payment types.

---

## Starting Payment

When ready to complete the transaction:

1. Review cart and total
2. Verify customer linked (if applicable)
3. Confirm fulfillment type
4. Click **"Pay"** or press **F8**

## Payment Troubleshooting

### Terminal Not Responding

1. Check terminal is powered on
2. Check cable connections
3. Try restarting terminal
4. Process as manual card entry (if authorized)
5. Contact IT if issue persists

### Card Read Errors

1. Clean card reader
2. Try different card orientation
3. Try chip instead of swipe (or vice versa)
4. Try contactless/tap
5. Manual entry if authorized

### Network Timeout

1. Wait 30 seconds
2. Do NOT re-process (may double-charge)
3. Check transaction status
4. Contact supervisor if uncertain

## Payment Security

### Best Practices

| Do | Don't |
|----|-------|
| Shield PIN pad | Look at customer's PIN |
| Process cards immediately | Leave card unattended |
| Clear sensitive data | Write down card numbers |
| Verify ID when required | Skip ID verification |

### Fraud Indicators

Be alert for:
- Multiple declined cards
- Nervous behavior
- Unusually large purchases
- Mismatched name/signature
- Purchasing high-theft items

**When suspicious:** Contact supervisor before completing transaction.
