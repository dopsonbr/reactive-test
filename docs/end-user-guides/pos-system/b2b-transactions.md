# B2B Transactions

Learn how to handle business customer transactions, account hierarchy, and net terms.

---

## B2B Overview

Business-to-Business (B2B) transactions differ from retail:

| Feature | Retail | B2B |
|---------|--------|-----|
| Pricing | Standard/sale | Contract pricing |
| Payment | At checkout | Net terms available |
| Delivery | Single address | Multiple locations |
| Ordering | Customer present | Often remote |
| Quantity | Small | Bulk orders |

## B2B Best Practices

### Before the Transaction

- Verify authorized buyer
- Confirm PO number and amount
- Check credit availability
- Confirm delivery addresses

### During the Transaction

- Apply contract pricing
- Document special requests
- Confirm all delivery dates
- Get PO signed off

### After the Transaction

- Send order confirmation
- Track all deliveries
- Follow up on delivery issues
- Ensure invoice sent correctly
